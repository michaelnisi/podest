AppDelegate

@UIApplicationMain
final class AppDelegate : UIResponder, UIApplicationDelegate

Receiving application events, AppDelegate is responsible for global things, namely triggering of background fetching, iCloud synchronization, and file downloading, providing a clear view over these central route entry points, close to their respective event sources.

Do not sync from anywhere else.
