Downloading

protocol Downloading : FileProxyDelegate

Downloading provides file facilities. For the most part, this API should not be used from the main queue, it might trap.

    Requests opportunistic background download of url if necessary.

  • preloadQueue(removingFiles:completionHandler:)

    Requests opportunistic background downloads of queued enclosures. It’s safe to call this ignorantly, it doesn’t produce redundant downloads. Furthermore, this method should keep the cache of downloaded files at a reasonable size.

    Declaration

    Swift

    func preloadQueue(removingFiles: Bool, completionHandler: ((Error?) -> Void)?)

    Parameters

    removingFiles

    Pass true to remove stale files.
    completionHandler

    The block to execute when preloading requests have been forwarded.

  • url(for:)

    Returns a file URL matching url if the resource has been downloaded or a remote URL if not, initiating an opportunistic background download. The returned URL might be used for AV streaming, the purpose of this API.

    Throws

    Throws FileRepository.MobileData if user defaults prevent streaming over mobile data and url isn’t available locally or over Wi-Fi.

    Declaration

    Swift

    func url(for url: URL) throws -> URL

    Parameters

    url

    The URL to access locally or remotely.

    Return Value

    A URL for AVPlayer.
